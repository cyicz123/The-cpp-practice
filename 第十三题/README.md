## 第一题
* 常对象需要利用构造函数初始化，初始化以后不能被改变。
* 类的常成员需要利用构造函数的初始化列表初始化（唯一方法），初始化后不能被改变。
* 任何函数都不能对常数据成员赋值。
* 如果有多个构造函数，都需要对常对象成员初始化列表赋值。

## 第二题实验截图
![13_2](http://photo.cyicz123.com/FvR9F1DUrT1U4vzBRYoBY3UJlHJH)
* 静态成员变量本质上是全局变量，即使一个对象都不存在，类的静态成员变量也存在。
* 静态成员函数实际上是全局函数。
* 设置静态成员这种机制的目的是将和某些类紧密相关的全局变量和函数写进类里面，看上去像一个整体，易于维护和理解。
* 必须在定义类的文件中对静态成员变量进行一次说明或者初始化，否则编译能过，链接无法通过。

## 第四题
友元分为友元函数和友元类两种。
1. 友元函数：一个类的友元函数可以访问该类的私有成员。
2. 如果A是B的友元类，那么A的成员函数可以访问B的私有成员。

|访问|public|protected|private|
| --|-------:|--------:|--------:|
|同一个类|yes|yes|yes|
|派生类|yes|yes|no|
|外部的类|yes|no|no|
